# DateR - Dating App
![text](https://i.ibb.co/S69WmMG/Screenshot-1-copy.png)

A dating website built using ASP.NET and Angular

## Built Using
- ASP.NET
- Angular
- Entity Framework
- Javascript - JWT
- [Toastr](https://github.com/CodeSeven/toastr)
- PostgreSQL
- Bootstrap
